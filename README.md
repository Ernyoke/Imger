# Imger

## Currently supported
* Grayscale
* Threshold (Binary, BinaryInv, Trunc, ToZero, ToZeroInv)
* Image padding (BorderConstant, BorderReplicate, BorderReflect)
* Convolution
* Blur (Average - Box, Gaussian)

## Merge into ```master```
```
git checkout master
git merge --no-commit --no-ff dev
```
